# n8n-nodes-vion

This is an n8n community node package. It provides the **Vion** integration for CardDAV and CalDAV interactions within [n8n](https://n8n.io).

Powered by the **Vion PIM Suite** (`@yamud/vobject`, `@yamud/carddav`).

## Features

* **Vion CardDAV:** Read, Create, Update and Delete contacts easily.
* **Type-Safe:** Uses the robust Vion parser engine.
* **Authentication:** Supports standard CardDAV credentials.

## Installation

Follow the [installation guide](https://docs.n8n.io/integrations/community-nodes/installation/) in the n8n documentation.

1. Go to **Settings > Community Nodes**.
2. Select **Install**.
3. Enter `@yamud/n8n-nodes-vion`.
